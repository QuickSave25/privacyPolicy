Privacy Policy for QuickSave

Data Storage

QuickSave stores all data locally on your device using Chrome's storage API. No data is sent to external servers.

Information We Store Locally

  • Text selections you choose to save
  • Screenshots you capture
  • Tags and organizational data
  • URLs of pages where content was saved

Chrome Permissions

QuickSave requires these permissions:

  • storage & unlimitedStorage: To save your content locally
  • activeTab: To interact with the current webpage
  • contextMenus: For the right-click save menu

Data Security

All data is stored locally on your device using Chrome's secure storage APIs. We do not collect, transmit, or store any personal information.
